# Scope

## In scope
- Single-player core game loop
- Seven-piece set
- Movement, rotation, gravity, locking
- Line clear
- Score and difficulty progression
- Minimal title screen
- Game over and restart
- One next piece preview
- Optional pause
- Simple local high score

## Out of scope
- Hold piece
- Ghost piece
- Multiplayer
- Multiple rulesets
- Stats-heavy modes
- Rich options menus
- Cosmetic skins/themes
- Experimental gimmicks

## Scope rule
A feature belongs only if it strengthens one of these:
- clarity
- play feel
- architectural integrity
- platform coherence
