# Vision

## Core statement
Quattro is a minimalist falling-blocks game for Commodore 64.
It aims to be historically sober in play experience and radically contemporary in development method.

## Desired outcome
The finished project should be:
- pleasant to play
- exemplary to read as software
- fully believable as C64 software

## Central promise
Quattro does not try to improve the genre by adding more.
It tries to make the essential unusually well.

## Design stance
No feature inflation.
No ironic remixing.
No decorative complexity.
Only what improves clarity, feel, and coherence.
