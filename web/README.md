# Quattro — GitHub Pages demo

Minimal browser demo for Quattro (this repository) using [VICE.js](https://github.com/rjanicek/vice.js). Experimental; not the primary way to play.

**To update the published demo:** from repo root run `make web-demo`, then commit the `web/` directory (including `quattro.prg` and `vendor/x64.js`). GitHub Pages must be set to deploy from this branch with source folder **/web**.
